# 📊 Blinkit Sales Analytics Dashboard

This project showcases an interactive sales analytics dashboard built using **Microsoft Power BI**, providing key insights into Blinkit's sales performance across various categories, locations, and time periods.

## 🗂 Project Files

- `Blinkit_Sales_Analytics_Dashboard.pbix` — The Power BI dashboard file.
- `data/` — Contains the dataset(s) used for the analysis.
- `screenshots/` — Preview images of the dashboard.
- `README.md` — Project documentation.

## 🎯 Project Objective

The goal of this dashboard is to:
- Track and analyze sales performance over time.
- Identify top-selling products and categories.
- Monitor regional and outlet-type contributions to total sales.
- Provide actionable insights for improving business decisions.

## 📊 Dashboard Features

- **KPIs**: Total Sales, Average Sales, Total Quantity, and Average Rating.
- **Category & Outlet Analysis**: Performance by product category and outlet type.
- **Location Insights**: Sales distribution across different locations.
- **Rating Distribution**: Customer ratings vs. sales correlation.
- **Trend Analysis**: Monthly and yearly sales trends.

## 🛠️ Tools & Technologies

- **Power BI** — For data modeling, visualization, and dashboard creation.
- **Excel / CSV** — Source data files.
- **DAX** — For calculated measures and custom metrics.

## 🚀 How to Use

1. **Download the Project**:
   - Clone the repository:
     ```bash
     git clone https://github.com/your-username/blinkit-sales-dashboard.git
     ```
   - Or download the `.zip` file and extract it.

2. **Open in Power BI**:
   - Install **Microsoft Power BI Desktop**.
   - Open `Blinkit_Sales_Analytics_Dashboard.pbix`.

3. **Explore**:
   - Navigate through different pages to explore sales trends, KPIs, and insights.
